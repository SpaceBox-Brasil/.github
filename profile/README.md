## 🇧🇷 Sobre a SpaceBox

A SpaceBox é uma comunidade brasileira open source focada no desenvolvimento de uma framework moderna baseada em Qbox para FiveM.

Nosso objetivo é construir uma estrutura profissional, organizada e otimizada para servidores RolePlay, mantendo compatibilidade com o ecossistema Qbox e toda a stack Ox.

Trabalhamos continuamente para desenvolver recursos modernos, seguros e escaláveis, trazendo uma experiência mais acessível e eficiente para a comunidade brasileira de FiveM.

A SpaceBox nasce com a proposta de criar uma framework open source preparada para projetos de grande porte, priorizando:

* Performance
* Segurança
* Código limpo
* Organização modular
* Compatibilidade
* Facilidade de deploy
* Ferramentas para desenvolvedores

Acompanhe nosso desenvolvimento e contribua conosco nesse projeto 🚀

## 🇧🇷 Sobre o Projeto SpaceBox

A SpaceBox é uma framework moderna baseada em Qbox, focada em entregar recursos de alta qualidade seguindo padrões modernos de desenvolvimento para FiveM.

Performance, segurança e organização são prioridades absolutas no projeto.

Nosso foco é desenvolver recursos preparados para servidores RolePlay modernos, mas também utilizáveis em diferentes tipos de servidores e projetos.

A framework está sendo construída utilizando tecnologias modernas da comunidade FiveM, incluindo:

* Qbox
* Ox Library
* Ox Inventory
* Ox Target
* PMA Voice
* Ox MySQL

Além de diversos recursos próprios desenvolvidos pela comunidade SpaceBox.

Para suporte, sugestões ou contribuições, entre em nosso servidor Discord.

---

## 🇺🇸 About SpaceBox Project

SpaceBox is a modern open source framework based on Qbox for FiveM.

Our goal is to provide a professional, scalable and optimized ecosystem for RolePlay servers while maintaining compatibility with the Qbox and Ox ecosystem.

Performance, security and clean code are extremely important for us.

We are focused on delivering high quality resources and modern development standards for the FiveM platform.

The project is currently under active development and community contributions are always welcome.

You can ask for support in our Discord server or follow the development through our GitHub organization.
